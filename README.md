## Environment vars

- `VITE_PUSHER_APP_ID`
- `VITE_PUSHER_KEY`
- `VITE_PUSHER_CLUSTER`
- `PUSHER_SECRET`

You can find them in Pusher's dashboard under Channels -> [app name] -> App Keys
